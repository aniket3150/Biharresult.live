Before You Start

  • Keep application/reference/certificate number ready before starting verification.
  • Select correct district, service type, and portal option to avoid wrong results.
  • Enter details exactly as per original document (no spelling/date mismatch).
  • Complete captcha carefully and submit only after re-checking all fields.
  • After verification, save screenshot/PDF/printout for official use and record.

How To Fill Form / Check Update

  1. Open the official ServiceOnline verification link.
  2. Login if required or continue with tracking/reference option.
  3. Enter application number, acknowledgement number, or service token.
  4. Fill captcha/security code and submit status request.
  5. Review current processing stage and remark from department.
  6. Check whether action is pending, approved, or rejected.
  7. Download/print status page and keep it for follow-up.
